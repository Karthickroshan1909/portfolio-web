# Portfolio Website on Azure

## Overview
This project demonstrates hosting a portfolio website on an Azure Virtual Machine using Apache Web Server.

## Technologies Used
- Azure Virtual Machine
- Ubuntu Linux
- Apache2
- HTML
- CSS
- Git
- GitHub

## Features
- Custom portfolio webpage
- Hosted on Apache Web Server
- Accessible via public IP address
- Managed through SSH

## Skills Practiced
- Linux Administration
- Azure VM Management
- Apache Configuration
- Web Hosting
- Git Version Control
